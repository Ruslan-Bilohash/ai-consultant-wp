# AI Consultant WP

**Сучасний AI-чатбот на базі Grok (xAI) для WordPress**

Плагін дозволяє легко додати на ваш сайт розумного віртуального консультанта, який працює 24/7, відповідає на питання клієнтів, збирає контакти та допомагає збільшувати продажі.

### Основні можливості

- Інтеграція з **Grok xAI API**
- Повністю кастомізований дизайн чату (кольори, градієнти, іконки)
- Гнучкий **системний промпт** — можеш налаштувати будь-яку роль бота
- Інтеграція зі **Telegram** (отримання повідомлень)
- Авто-відкриття чату з налаштуванням затримки
- Плаваючий віджет з вибором позиції
- Перевірка оновлень прямо в адмінці
- Адаптивний дизайн та висока безпека

### Встановлення

1. Завантажте репозиторій або ZIP-архів
2. Встановіть через **WordPress → Плагіни → Додати новий → Завантажити плагін**
3. Активуйте плагін
4. Перейдіть у меню **AI Consultant WP → Налаштування**
5. Вставте свій **Grok xAI API ключ** і налаштуйте промпт

### Отримання Grok API ключа

Перейдіть на [console.x.ai](https://console.x.ai), увійдіть через акаунт X і створіть новий API ключ.
