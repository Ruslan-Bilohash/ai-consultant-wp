# AI Consultant

**Modern Grok (xAI) powered chatbot for WordPress** with real-time Telegram notifications.

### Features
- Powered by **Grok xAI** (latest model)
- Real-time notifications to **Telegram**
- Fully customizable design (colors, gradients, position, icon)
- Live preview in admin panel
- Conversation history saved
- Responsive and beautiful UI

### Installation

1. Download the latest release
2. Upload to `/wp-content/plugins/ai-consultant/`
3. Activate the plugin
4. Go to **AI Consultant → Settings**
5. Enter your Grok xAI API Key
6. Save and enjoy!

### Requirements
- WordPress 6.4+
- PHP 7.4+
- Grok xAI API key (get it at [console.x.ai](https://console.x.ai/))
