**AI Consultant WP** — сучасний інтелектуальний чатбот на базі Grok (xAI) для WordPress 6.9 UA.

Плагін дозволяє додати на сайт розумного AI-асистента, який:
- Відповідає клієнтам 24/7
- Консультує щодо послуг
- Збирає ліди та контакти
- Підтримує інтеграцію з Telegram

### Основні можливості:
- Повністю кастомізований дизайн чату
- Гнучкий системний промпт (можна описати будь-яку роль бота)
- Інтеграція з Grok xAI API
- Сповіщення в Telegram
- Авто-відкриття, адаптивність, безпека
- Легка перевірка оновлень прямо з адмінки

Ідеально підходить для бізнесів, які хочуть автоматизувати консультації та підвищити продажі.
